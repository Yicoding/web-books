# css 实现文字省略号

CSS 实现文字省略号有多种方式，根据行数需求可以选择不同的实现方法。以下是常用的实现方法：

`推荐方案1`

## 1. 单行省略号

通过设置 `text-overflow` 属性实现单行文本省略。

```jsx
/**
 * defaultShowCode: true
 */

import './demo1.less';

export default () => (
  <div className="ellipsis">
```
